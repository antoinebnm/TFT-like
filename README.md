# Auto Battler Game

Auto Battler (Auto-chess) game made with [Unity 6.3 LTS](https://docs.unity3d.com/6000.3/Documentation/Manual/index.html).

## Run Locally

Clone the project

Import it into Unity (`Add a project from Disk`) or using `Add from repository` somehow.
